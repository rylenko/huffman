Implementation of Huffman's compression / decompression algorithms.

<h1 align="center">Make</h1>

To build:

```
$ make
```

To install:

```
$ sudo make install
```

To uninstall:

```
$ sudo make uninstall
```

To clean:

```
$ make clean
```

<h1 align="center">Usage</h1>

```
$ huffman compress <file> > out
```

```
$ huffman decompress <file> > out
```
